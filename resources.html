Reporting Scams & Security Issues

If you encounter a scam or security issue:

  1. Document Everything: Take screenshots, save messages, record wallet addresses
  2. Report to Pi Network: Use official reporting channels within the app
  3. Warn the Community: Share details (without personal info) on official forums
  4. Report to Platforms: Report fake accounts to social media platforms
  5. Local Authorities: For significant fraud, file a police report
  6. Learn and Share: Use the experience to help educate others
